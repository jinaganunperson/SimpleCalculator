# (C# 코딩) 나만의　계산기
 ## 개요
- C# 프로그래밍학습
- 1줄 소개 : 사칙연산을　수행하는　계산기　프로그램을　만들어　봅니다．
- 사용한 플랫폼: -C#, .NET Windows Forms, Visual Studio, GitHub
- 사용한 컨트롤 : -Label, TextBox, ListBox, Button
- 사용한 기술과 구현한 기능　：　
- Visual Studio를이용하여UI 디자인
- string 클래스를이용한사용자입력데이터처리
- DateTime클래스를이용한현재시간정보구하기
- 수업중에　배우고　사용했던　클래스들　관련된　설명


- 실습중에　구현한　기능들　설명
- 숫자　입력　기능　：　숫자　버튼　클릭　시　텍스트북에　숫자를　표시함．
- 사칙연산　계산　기능　：　２개의　피연산자의　입력값을　ｉｎｔ로　바꾸어　더하기　계산을　수행하고　그　결과를　저장합니다．
- 계산　결과　출력　：　계산　결과　값을　문자열로　변환하여　표시합니다．
- ｃ버튼　：　현재의　모든　내용을　삭제하고　처음　（초기화된）상태로　되돌아갑니다．
- ｃｅ버튼　：　현재　입력된　값을　삭제하고　초기화된　상태로　되돌아갑니다．
- ｄｅｌ버튼　：　현재　입력된　값의　마지막　문자를　삭제합니다．　１００입력　후에　ｄｅｌ버튼을　눌렀다면　１０으로　변경됩니다．

## 실행화면(과제1)
- 과제1코드의실행스크린샷
![과제1 실행화면](img/screenshot-1.png)

- 과제내용
- ＵＩ구성　：　텍스트박스（입력　표시，　결과　표시），　버튼（계산）　등을　적절히　배치합니다．
- 숫자　입력　기능　：　숫자　버튼　클릭　시　텍스트북에　숫자를　표시하도록　합니다．
- 사칙연산　계산　기능　：　２개의　피연산자의　입력값을　ｉｎｔ로　바꾸어　더하기　계산을　수행하고　그　결과를　저장합니다．
- 계산　결과　출력　：　계산　결과　값을　문자열로　변환하여　표시합니다．

- 구현내용과　기능설명
- 텍스트박스에　입력값과　결과가　표시됩니다．　
- 계산　버튼을　누르면　입력　내용을　기반으로　계산합니다．
- 숫자　버튼을　누르면　해당　숫자가　텍스트박스에　추가됩니다．
- 사칙　연산　계산　기능으로　피연산자의　입력값으로　더하기　계산을　수행하여　그　결과를　저장합니다．
